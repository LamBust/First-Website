Библиотека math — инструменты для математических вычислений в Python Библиотека math входит в стандартную библиотеку Python и используется для выполнения математических операций. Она предоставляет готовые функции, которые позволяют выполнять вычисления быстрее и точнее, чем при использовании обычных формул. math работает с вещественными числами и активно применяется в задачах по математике, физике, информатике и инженерии. С помощью этой библиотеки можно находить квадратные корни, степени, логарифмы, а также использовать тригонометрические функции (синус, косинус, тангенс). Кроме того, в math содержатся важные математические константы, такие как π и e. Пример использования: import math a = 9 b = 2 print(math.sqrt(a)) # квадратный корень print(math.pow(a, b)) # возведение в степень print(math.pi) # число π print(math.cos(0)) # косинус
